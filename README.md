# BTK Akademi

This repository contains the Onepage graduation project made for BTK-Akademi Front end training bootcamp. </br>

## Requirements and Technologies Used
 - Visual Studio Code
 - Html5
 - CSS3 
 - Javascript 
 - Bootstrap5 
 - React Js 
 - Git 
 - Gitlab, Github 
 - API
 - Postman
 - git push -u github master

## Application Features

 - One page pattern
 - Login / Register 
 - I18N özelliği 
 - Axios libraries
 - Router libraries 
 - Bootstrap5 
 - FontAwesome 
 - Google Font (Poppins) 
 - ScrollSpy 
 - Back to Top

I thank [@hamitmizrak](https://github.com/hamitmizrak/) and [@BTKAkademi](https://www.btkakademi.gov.tr/) for their contributions.
